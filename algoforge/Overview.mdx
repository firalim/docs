## 🔥 What is AlgoForge?

**AlgoForge** is the world’s first **on-chain AI content guild** that lets anyone — developer, poet, marketer, or meme lord — create content with AI *and* instantly verify it via blockchain.

We’re not just generating text. We’re forging:

- 🎙️ Voice-confirmed limericks  
- 🧬 On-chain micro-ads  
- 📺 AI-generated video messages  
- 🎮 Wacky game lore and character bios  
- 📡 Content agents that evolve with every trigger  

All of it powered by:
- ⚙️ **Bolt.new** (no-code + code hybrid for insane dev speed)  
- 🧠 **ElevenLabs** (realistic AI voices = emotional validation)  
- 🎥 **Tavus Video** (turn prompts into humanized face-to-cam video)  
- 💾 **Supabase** (for state and auth)  
- ⛓️ **Algorand Blockchain** (for proof, permanence, and programmable autonomy)

And yes — it all runs automatically. Push prompt, mint output, publish story.  
*No middlemen. No gatekeepers. Just autonomous creativity.*

## 🔑 Key Features

### 🎙️ Voice-Backed Content Generation  
Every text is confirmed by **AI-generated voice** (via ElevenLabs), creating emotional + human trust in machine output.

> It’s not just content. It’s *spoken* proof of intent.

---

### 🔗 On-Chain Verification via Algorand  
Every creation is stored or anchored on-chain with cryptographic trust.  
You don’t just own the content — the blockchain **proves** you made it.

---

### 🤖 Autonomous Agent Framework  
Use prompts to spin up autonomous agents that can:
- Generate themed content
- Respond to trends
- Publish at intervals
- Mutate with logic over time

This is **programmable storytelling**, not static content.

---

### 📺 Prompt-to-Video Pipeline  
With **Tavus AI**, creators can turn text prompts into **face-to-camera videos** with consistent avatars, voices, and emotion.

- Great for marketers, founders, creators, and meme influencers.
- All wrapped in a blockchain timestamp.

---

### ⚡ Built with Bolt.new  
Bolt enables **speedrun development** with powerful internal state logic, agent configuration, and no-code deploy options.  
All hosted, fast, and connected to your Supabase backend in minutes.

---

### 🧱 Modular Stack Design  
- **Supabase** for authentication, state, and agent memory  
- **Algorand** for trustless logs and minting  
- **ElevenLabs** for emotion & trust  
- **Tavus** for video immersion  
- **Bolt.new** for dev speed & agent control

Every layer modular. Every agent programmable.
