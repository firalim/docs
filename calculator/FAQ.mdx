      <Accordion title="📊 IRR vs NPV vs ROI - What's the difference?" icon="calculator">
        <CardGroup cols={3}>
          <Card title="📈 ROI" icon="trending-up" color="#10B981">
            Return on Investment

            `(Gain - Cost) / Cost`

            ✅ Quick comparisons\
            ❌ Ignores time value
          </Card>
          <Card title="⏰ IRR" icon="clock" color="#8B5CF6">
            Internal Rate of Return

            `Rate where NPV = 0`

            ✅ Time-adjusted returns\
            ❌ Can have multiple solutions
          </Card>
          <Card title="💰 NPV" icon="dollar-sign" color="#F59E0B">
            Net Present Value

            `Present Value - Investment`

            ✅ Absolute value creation\
            ❌ Requires discount rate
          </Card>
        </CardGroup>
        <Info>
          Our calculators show all three metrics with interactive visualizations at [fc.firuz-alimov.com](https://fc.firuz-alimov.com)\!
